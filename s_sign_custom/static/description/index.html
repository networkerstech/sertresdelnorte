SUITEDOO | Document Sign

Generate signing requests from specific models and dynamically manage configuration.

Features.
  • Customize any PDF document sign.
  • Customize any PDF Odoo report document sign.
  • Customize dinamically the signers for documents.
  • Send customizable messages via email to notify the sign requests.
  • Signed report is attach to signer model object.

Generate a signature request for custom model sending an email like Odoo Sign Native app.
When the report is signed this is attached to signer model.

Go to Sign main module and select UPLOAD A PDF TEMPLATE

Customize your signature document.

Go to created Signature Template Properties .

Dynamic Signature Configuration Fields
  1. Check Dynamic configuration to set the template model configuration.
  2. Template types: Report or Document
  3. Action report: Dynamic menu action showed in model Print menu.
  4. Report Model: Select the model to set the signature request.
  5. Report: Select the report to sign.
  6. Signers: Select the field that refers to the contact who will sign the report.\n Select the "ID" field to indicate that the person who will sign the document is the record contact.

If you select report or document template type

  • Report: is used for PDF report in selected model.
  • Document: is used for any PDF document and it will be associated to selected model.
